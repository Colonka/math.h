In this project I\textquotesingle{}ve developed my own version of the standard math.\+h library in the C programming language. This library implements basic mathematical operations, which are then used in various algorithms.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Information}{Information}}\label{md__r_e_a_d_m_e_autotoc_md2}
C mathematical operations are a group of functions in the standard library of the C programming language implementing basic mathematical functions. All functions use floating-\/point numbers in one manner or another. Different C standards provide different, albeit backwards-\/compatible, sets of functions. Any functions that operate on angles use radians as the unit of angle.\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysection{\texorpdfstring{Instruction for running}{Instruction for running}}\label{md__r_e_a_d_m_e_autotoc_md3}
Being in math.\+h/src folder run command {\ttfamily make} in terminal that create static library s21\+\_\+math. Then include \doxylink{s21__math_8h}{s21\+\_\+math.\+h} in your file and link s21\+\_\+math while compilation of your source files.
